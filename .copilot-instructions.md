# HTXV2 - Cryptocurrency Trading Platform - Copilot Instructions

## Project Overview

HTXV2 is a comprehensive GCP-based cryptocurrency trading platform with ML/AI capabilities. The system combines real-time trading, portfolio management, data analytics, and intelligent trading signals powered by local FinGPT models and cloud AI services.

**Example API Endpoint:**
```python
from fastapi import APIRouter, Depends, HTTPException, status
from sqlalchemy.ext.asyncio import AsyncSession
from app.db.session import get_db
from app.schemas.trading import OrderCreate, OrderResponse
from app.services.trading import TradingService

router = APIRouter(prefix="/api/v1/trading", tags=["trading"])

@router.post("/orders", response_model=OrderResponse)
async def create_order(
    order: OrderCreate,
    db: AsyncSession = Depends(get_db),
    current_user: User = Depends(get_current_user),
) -> OrderResponse:
    """Create a new trading order."""
    try:
        trading_service = TradingService(db)
        result = await trading_service.create_order(order, current_user.id)
        return result
    except ValueError as e:
        raise HTTPException(
            status_code=status.HTTP_400_BAD_REQUEST,
            detail=str(e)
        )
```

**Example AI Service:**
```python
class LLMService:
    def __init__(self):
        self.providers = [
            FinGPTProvider(),
            VertexAIProvider(),
            OpenAIProvider()
        ]
    
    async def generate_trading_signal(
        self, 
        market_data: MarketData,
        user_context: UserContext
    ) -> TradingSignal:
        """Generate trading signal with fallback providers."""
        for provider in self.providers:
            try:
                if await provider.is_healthy():
                    return await provider.generate_signal(market_data, user_context)
            except Exception as e:
                logger.warning(f"Provider {provider.name} failed: {e}")
                continue
        
        raise LLMServiceUnavailableError("All AI providers are unavailable")
```
