# Token Economy (ClaudeAI)

Правила минимизации токенов для ассистента ClaudeAI.

- Отвечать кратко (≤ 3–4 строки). Без отчётов, резюме и больших цитат.
- Фиксировать результат только в журнале: `docs/journal/YYYY-MM-DD.md` (1–3 пункта + ссылка на PR/коммит).
- Не вставлять большие блоки кода/логи — давать ссылки на файлы/диффы.
- Мелкие правки объединять в один коммит с 1‑строчным описанием.
- Не инициировать пуши, требующие токенов (LLM/внешние вызовы) в CI: включён `token-guard`.
- Для предварительного просмотра использовать Storybook/скриншоты‑артефакты (а не вставки картинок в ответы).

Скрипты
- `scripts/journal-note.sh "msg"` → добавляет заметку в `docs/journal/YYYY-MM-DD.md`.
- `scripts/journal-note.ps1 -Message "msg"` → то же под Windows.
- `scripts/ci/llm-usage-check.sh` → проверка, что в репозитории нет шагов, тратящих токены LLM (используется в CI и pre‑push hook).
