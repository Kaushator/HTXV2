## 2. Инфраструктура проекта

### 2.1. Компоненты системы
Проект состоит из следующих компонентов:
- **Backend**: API сервер на Python (Flask/FastAPI)
- **Frontend**: Веб-интерфейс на React/Vue
- **FinGPT**: Сервис для обработки данных с использованием NLP/ML
- **Инфраструктура**: Terraform конфигурация для автоматизированного развертывания

### 2.2. Архитектура развертывания
Все компоненты развертываются как сервисы в Google Cloud Run:
- Каждый сервис работает в контейнере Docker
- Образы хранятся в Google Artifact Registry
- Сервисы связываются через API
- Процесс CI/CD автоматизирован через GitHub Actions
